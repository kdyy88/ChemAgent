# 总体生化目标
以泽布替尼（zanubrutinib）为参考，通过骨架跃迁（scaffold hopping）生成并筛选 1 个结构相似但 Murcko 骨架不同的新分子，并给出与参考分子的结构差异标注（基于 MCS/取代基对照）。

# 执行管线 (Pipeline)
## 阶段 1：参考分子标准化与权威结构获取
* **动作意图**: 从公共数据库检索泽布替尼的权威结构记录并标准化为可计算的 canonical SMILES/InChIKey，作为后续相似性、骨架与差异比对的唯一基准。
* **依赖工件 (Inputs)**: 无
* **挂载工具 (Required Tools)**: database-lookup, rdkit
* **预期产出 (Outputs)**:
  - 泽布替尼的标准化结构（canonical SMILES、InChIKey、主要数据库来源：PubChem CID / ChEMBL ID）
  - 去盐/中性化（如适用）后的 cleaned SMILES（用于后续所有计算）

## 阶段 2：参考分子骨架解析与可保留药效团约束抽取
* **动作意图**: 计算参考分子的 Murcko scaffold、关键官能团与可反应“战头”(warhead)片段（如存在）位置，用于定义骨架跃迁时的保留/替换边界。
* **依赖工件 (Inputs)**: 阶段1输出的 cleaned SMILES
* **挂载工具 (Required Tools)**: rdkit
* **预期产出 (Outputs)**:
  - 参考分子的 Murcko scaffold（SMILES）与取代基分解映射（每个连接点的取代基 SMILES）
  - 关键官能团/药效团约束清单（例如：保持共价受体片段、HBD/HBA 布局、芳环/杂芳环平面片段等；以 SMARTS/子结构方式表达）

## 阶段 3：数据库驱动的骨架跃迁候选获取（相似但骨架不同）
* **动作意图**: 以参考分子的结构为查询，在公共数据库中检索相似分子集合，并通过“Murcko scaffold 不同”筛选实现骨架跃迁候选池。
* **依赖工件 (Inputs)**: cleaned SMILES；阶段2输出的 Murcko scaffold 与药效团约束
* **挂载工具 (Required Tools)**: database-lookup, rdkit
* **预期产出 (Outputs)**:
  - 候选分子列表（至少包含：候选 canonical SMILES、数据库 ID/来源、与参考的指纹相似度、候选 Murcko scaffold）
  - 已应用的筛选规则记录：相似度阈值、骨架不同判据（Murcko scaffold SMILES 不同或 scaffold 指纹距离）与药效团/子结构约束是否满足

## 阶段 4：候选评估与单一新分子定稿
* **动作意图**: 对候选进行可行性评估（相似性、理化性质、结构有效性），选择 1 个作为“类似泽布替尼的新分子”并给出标准化结构。
* **依赖工件 (Inputs)**: 阶段3候选列表
* **挂载工具 (Required Tools)**: rdkit
* **预期产出 (Outputs)**:
  - 定稿新分子的标准化结构（final_smiles、InChIKey）
  - 与泽布替尼的相似性摘要（Tanimoto；使用明确的指纹类型，如 Morgan/ECFP）
  - 关键理化描述符摘要（例如：MW、cLogP(如可得/可计算)、HBD/HBA、tPSA、RotB、环数；用于解释“类似性”与可药性边界）

## 阶段 5：与泽布替尼的结构差异标注与可视化对照
* **动作意图**: 基于最大公共子结构（MCS）/子结构匹配生成“差异清单”，将变化定位到骨架与取代基层面，输出人类可读的差异描述与（如支持）高亮对照图。
* **依赖工件 (Inputs)**: 泽布替尼 cleaned SMILES；定稿新分子 final_smiles
* **挂载工具 (Required Tools)**: rdkit
* **预期产出 (Outputs)**:
  - differences_summary：骨架变化说明（旧 scaffold vs 新 scaffold）与每个连接点的取代基差异（新增/删除/替换官能团）
  - MCS 对照结果：MCS SMARTS/原子映射、差异原子集合（用于高亮）
  - （如工具支持绘图）带差异高亮的 2D 对照图及对应文字说明

# 关键数据缺口
- “类似”的成功标准未明：需要确认相似性度量与阈值（例如 ECFP4 Tanimoto ≥ 0.5/0.6）及必须保留的作用机制（例如是否必须保留共价 warhead / 关键氢键受体-供体拓扑）。
- 若有特定靶点/选择性约束（例如 BTK vs 其他激酶）、或理化性质窗口（MW、tPSA、cLogP、可溶性方向），需提供以便在阶段3/4设置硬过滤条件。
